오늘 푼 문제는 코드+주석으로 정리하여 git으로 그날 바로 올립니다.  
쌓이는 문제 풀이 수 만큼 실력도 향상되어 이후 코딩테스트에서 꼭 올솔(all solve)을 해보고 싶습니다😆
